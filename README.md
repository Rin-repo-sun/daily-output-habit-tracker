# 毎日アウトプット習慣トラッカー

毎日のアウトプットを記録し、習慣化をサポートするWebアプリケーションです。

![Daily Output Habit Tracker](https://images.unsplash.com/photo-1484480974693-6ca0a78fb36b?w=800&q=80)

## 🌟 特徴

- **📅 カレンダービュー**: 月間カレンダーで過去のアウトプットを一目で確認
- **🔥 連続記録日数**: 現在の連続記録（ストリーク）を追跡
- **📊 統計情報**: 今月のアウトプット数、総アウトプット数を表示
- **💾 ローカル保存**: ブラウザのLocalStorageを使用してデータを保存
- **🎨 美しいUI**: Tailwind CSSによるモダンでレスポンシブなデザイン
- **✨ モチベーション**: 連続記録に応じた励ましのメッセージ

## 🚀 デモ

[Live Demo](https://github.com/Rin-repo-sun/daily-output-habit-tracker) *(デプロイ後にリンクを更新)*

## 💡 使い方

1. カレンダーから日付を選択
2. その日のアウトプット内容を入力（学んだこと、作ったもの、気づいたことなど）
3. 「保存する」ボタンをクリック
4. カレンダーに記録が表示され、連続記録日数が更新されます

### 主な機能

#### 📈 統計ダッシュボード
- **現在の連続記録**: 連続でアウトプットした日数
- **今月のアウトプット**: 今月記録した回数
- **総アウトプット数**: これまでの総記録数

#### 📆 カレンダー表示
- 月間カレンダーでアウトプットした日を視覚的に確認
- アウトプット済みの日はオレンジ〜ピンクのグラデーションで表示
- チェックマークアイコンで完了を明示

#### ✍️ アウトプット記録
- 各日のアウトプット内容をテキストで保存
- 後から編集・削除が可能
- 今日の日付は「今日」ラベルで識別しやすく

## 📦 インストール

```bash
# リポジトリをクローン
git clone https://github.com/Rin-repo-sun/daily-output-habit-tracker.git

# ディレクトリに移動
cd daily-output-habit-tracker

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

## 🏗️ ビルド

```bash
# プロダクションビルド
npm run build
```

ビルドされたファイルは `dist` ディレクトリに出力されます。

## 📱 レスポンシブデザイン

デスクトップ、タブレット、モバイルデバイスに対応しています。

## 💾 データの保存

すべてのデータはブラウザのLocalStorageに保存されます。
- データはブラウザ内にのみ保存され、外部には送信されません
- ブラウザのキャッシュをクリアするとデータが削除されますのでご注意ください
